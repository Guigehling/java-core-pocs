# Reflection API POC

POC desenvolvida em Java 21 com o objetivo de explorar os principais conceitos da Reflection API.

## Objetivos

Durante esta POC foram estudados os seguintes conceitos:

- Leitura de metadados de classes em runtime
- Identificação de atributos e métodos
- Invocação dinâmica de métodos públicos e privados
- Modificação de atributos privados via reflection
- Manipulação de `Field` e `Method`
- Tratamento de exceções da Reflection API

## Conceitos abordados

- `Class`
- `Field`
- `Method`
- `getDeclaredFields`
- `getDeclaredMethods`
- `getDeclaredMethod`
- `invoke`
- `setAccessible`
- `Reflection API`

## Tecnologias

- Java 21
