# Properties API POC

POC desenvolvida em Java 21 com o objetivo de explorar os principais recursos da `Properties API` da linguagem.

## Objetivos

Esta POC demonstra:

- Criação de propriedades utilizando `java.util.Properties`
- Persistência de propriedades em arquivo `.properties`
- Leitura de propriedades a partir do classpath
- Manipulação de System Properties da JVM
- Utilização de operações de IO com arquivos

## Conceitos abordados

- `Properties`
- `setProperty`
- `putAll`
- `store`
- `load`
- `System.getProperty`
- `System.setProperty`
- `InputStream`
- `FileOutputStream`
- `ClassLoader`

## Tecnologias

- Java 21
